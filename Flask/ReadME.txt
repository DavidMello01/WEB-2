Nome do Framework: Flask
Ano de Lançamento: 2010
Desenvolvedor: Pocoo Team
Linguagem de Programação: Python
Quando é Recomendado Utilizar o Framework: Flask é recomendado para projetos web pequenos a médios, 
onde a simplicidade e flexibilidade são prioridades.

Principais Características -> 
	- Simplicidade
	- Flexibilidade
	- Extensibilidade
